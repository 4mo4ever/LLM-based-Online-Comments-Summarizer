# LLM-based-Online-Comments-Summarizer
Project Overview: LLM-Based Online Comments Summarizer
Background & Origin
This project is developed as a collaborative group assignment for the CS6-2 Capstone course at the University of Sydney (USYD).

What It Is
It is an intelligent backend system powered by the MiniMax Large Language Model and orchestrated using LangGraph. It is designed to automatically process and analyze large volumes of online comments from platforms like YouTube and Reddit.

Project Goal
The primary goal of this system is to save time for content creators, analysts, and businesses. Instead of manually reading thousands of comments, users can quickly understand the core opinions, emotional tone, and demographic makeup of their audience through structured data.

Key Features

1. Persona Extraction: Automatically identifies the behavioral traits and domain expertise level of the commenter (e.g., distinguishing a "professional reviewer" from a "casual user").

2. Sentiment Analysis: Evaluates the emotional polarity of each comment on a scale from 0.0 (negative) to 1.0 (positive).

3. Smart Impact Weighting: Calculates a final "weighting score" for each comment based on algorithmic factors like the number of likes, how recent the comment is, and the author's expertise.

4. Multi-Agent Pipeline: Uses a modular, parallel AI agent architecture to ensure fast and accurate data processing.
